# Europe Odoo Integration

A FastAPI-based integration for Odoo, designed to streamline delivery date management and order processing.

## Features
- Flexible Date Support: Handles dual date formats (DD/MM/YYYY and YYYY-MM-DD).
- Endpoint Optimization: Enhanced delivery date endpoints for better performance.
- Payload-driven: Efficient order ID handling within the request payload.

## Tech Stack
- Language: Python
- Framework: FastAPI
- ERP: Odoo Integration

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/marcelo-fg/Europe_Odoo.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the FastAPI application:
```bash
uvicorn my_odoo_api:app --reload
```
